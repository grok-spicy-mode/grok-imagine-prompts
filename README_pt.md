## O que é Grok Imagine?

Grok Imagine é o poderoso modelo de geração de imagens da X.AI integrado à plataforma Grok. Ele cria imagens de alta qualidade a partir de descrições de texto usando tecnologia de IA avançada. A plataforma oferece dois modos:

- **Modo Normal**: Geração de imagens padrão adequada para todos os públicos
- **Spicy Mode**: Modo avançado com menos restrições de conteúdo para expressão criativa

## Como Usar Grok Imagine Spicy Mode

### Passo 1: Acessar Grok Imagine
1. Visite [GrokPrompts.app](https://grokprompts.app) para geração instantânea de prompts
2. Ou acesse X.com e entre no Grok
3. Navegue até a aba "Imagine"

### Passo 2: Ativar Spicy Mode
- Ative o botão "Spicy Mode" na interface
- Nota: spicy mode está disponível apenas para usuários verificados

### Passo 3: Usar Prompts deste Repositório
- Navegue pelas nossas [categorias de prompts](#categorias-de-prompts)
- Copie qualquer prompt e cole no Grok Imagine
- Modifique usando nossos [modificadores de estilo](#folha-de-dicas-de-modificadores-de-estilo)
- Gere e itere

### Dicas Profissionais:
- Comece com assuntos específicos e adicione modificadores de estilo
- Use indicadores de resolução: "8k", "4k uhd", "high resolution"
- Combine múltiplos estilos: "cinematic lighting, oil painting style"
- Experimente com prompts negativos para refinar resultados

## Folha de Dicas de Modificadores de Estilo

Melhore qualquer grok imagine prompts com estes modificadores comprovados:

| Categoria | Modificadores | Efeito |
|----------|-----------|--------|
| **Qualidade** | 8k uhd, 4k, high resolution, hyperdetailed, intricate details | Aumenta o detalhe e nitidez da imagem |
| **Iluminação** | cinematic lighting, dramatic lighting, golden hour, rim lighting, volumetric lighting | Controla o humor e atmosfera |
| **Câmera** | depth of field, bokeh, wide angle, macro shot, telephoto | Simula efeitos de câmera |
| **Estilo de Renderização** | octane render, unreal engine, photorealistic, ray tracing | Qualidade de renderização profissional |
| **Artístico** | trending on artstation, award winning, masterpiece, by [artist name] | Melhora a qualidade artística |
| **Cor** | vibrant colors, muted tones, black and white, neon colors | Controle de paleta de cores |

Consulte nossa [Referência Completa de Modificadores de Estilo](./guides/style-modifiers-reference.md) para mais de 100 modificadores.

## Fórmula de Prompts

Use esta fórmula comprovada para resultados consistentes:

```
[Sujeito] + [Ação/Pose] + [Ambiente] + [Iluminação] + [Estilo] + [Modificadores de Qualidade]
```

**Exemplo:**
```
Um guardião místico da floresta + em pé ao lado de um carvalho antigo +
floresta encantada com cogumelos brilhantes + iluminação etérea suave +
estilo de arte digital fantástica + 8k uhd, hyperdetailed, trending on artstation
```

Para mais modelos e fórmulas, visite nosso [Guia de Fórmulas de Prompts](./guides/prompt-formula.md).

## FAQ

### O que são grok imagine prompts?
Grok imagine prompts são descrições de texto que guiam a IA do Grok para gerar imagens específicas. Eles podem variar de simples ("um pôr do sol sobre montanhas") a prompts complexos com múltiplos modificadores.

### Como grok imagine spicy mode difere do modo normal?
Spicy mode permite mais liberdade criativa com menos restrições de conteúdo. É projetado para expressão artística e temas maduros.

### Onde posso encontrar mais grok prompts for images?
Este repositório contém mais de 200 prompts em 10 categorias. Você também pode gerar prompts personalizados em [GrokPrompts.app](https://grokprompts.app).

### O que torna um bom grok imagine prompt?
Bons prompts são específicos, usam modificadores de estilo, incluem palavras-chave de qualidade (8k, detailed) e especificam iluminação/atmosfera.

### Posso usar estes prompts comercialmente?
Sim, estes prompts são licenciados sob MIT. Imagens geradas seguem os termos de serviço do Grok.

### Como obtenho melhores resultados?
- Seja específico sobre o assunto e detalhes
- Use múltiplos modificadores de estilo
- Especifique iluminação e atmosfera
- Inclua palavras-chave de resolução/qualidade
- Itere e refine com base nos resultados

### Onde posso compartilhar meus resultados?
Junte-se às discussões no Reddit (r/xAI) ou compartilhe no X com #GrokImagine. Visite [GrokPrompts.app](https://grokprompts.app) para recursos da comunidade.
