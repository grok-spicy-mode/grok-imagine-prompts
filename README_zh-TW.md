## 什麼是Grok Imagine？

Grok Imagine是X.AI整合到Grok平台的強大圖像生成模型。它使用先進的AI技術從文字描述創建高品質圖像。平台提供兩種模式：

- **普通模式**：適合所有受眾的標準圖像生成
- **Spicy Mode**：內容限制更少的進階模式，用於創意表達

## 如何使用Grok Imagine Spicy Mode

### 第1步：存取Grok Imagine
1. 造訪[GrokPrompts.app](https://grokprompts.app)進行即時提示詞生成
2. 或前往X.com並進入Grok
3. 導航到「Imagine」標籤

### 第2步：啟用Spicy Mode
- 在介面中開啟「Spicy Mode」開關
- 注意：spicy mode僅適用於已驗證使用者

### 第3步：使用本儲存庫的提示詞
- 瀏覽我們的[提示詞分類](#提示詞分類)
- 複製任何提示詞並貼上到Grok Imagine
- 使用我們的[風格修飾符](#風格修飾符速查表)進行修改
- 生成並迭代

### 專業提示：
- 從具體主題開始並添加風格修飾符
- 使用解析度提示：「8k」、「4k uhd」、「high resolution」
- 組合多種風格：「cinematic lighting, oil painting style」
- 使用負面提示詞實驗以優化結果

## 提示詞公式

使用這個經過驗證的公式獲得一致的結果：

```
[主題] + [動作/姿勢] + [環境] + [光照] + [風格] + [品質修飾符]
```

**範例：**
```

有發光蘑菇的魔法森林 + 柔和的空靈光照 +
奇幻數位藝術風格 + 8k uhd, hyperdetailed, trending on artstation
```

## 常見問題

### 什麼是grok imagine prompts？
Grok imagine prompts是指導Grok的AI生成特定圖像的文字描述。它們可以從簡單的（「山上的日落」）到具有多個修飾符的複雜提示詞。

### grok imagine spicy mode與普通模式有何不同？
Spicy mode允許更多的創意自由，內容限制更少。它專為藝術表達和成熟主題設計。

### 在哪裡可以找到更多grok prompts for images？
此儲存庫包含10個類別中的200多個提示詞。您還可以在[GrokPrompts.app](https://grokprompts.app)上生成自訂提示詞。

### 什麼使一個grok imagine prompt變好？
好的提示詞是具體的，使用風格修飾符，包含品質關鍵詞（8k、detailed），並指定光照/氛圍。

### 我可以商業使用這些提示詞嗎？
可以，這些提示詞採用MIT授權。生成的圖像遵循Grok的服務條款。

### 如何獲得更好的結果？
- 對主題和細節要具體
- 使用多個風格修飾符
- 指定光照和氛圍
- 包含解析度/品質關鍵詞
- 根據結果迭代和優化

### 我可以在哪裡分享我的結果？
加入Reddit（r/xAI）上的討論或在X上使用#GrokImagine分享。造訪[GrokPrompts.app](https://grokprompts.app)獲取社群功能。
