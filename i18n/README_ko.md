## Grok Imagine이란?

Grok Imagine은 Grok 플랫폼에 통합된 X.AI의 강력한 이미지 생성 모델입니다. 고급 AI 기술을 사용하여 텍스트 설명에서 고품질 이미지를 생성합니다. 플랫폼은 두 가지 모드를 제공합니다:

- **일반 모드**: 모든 대상에게 적합한 표준 이미지 생성
- **Spicy Mode**: 창의적 표현을 위한 콘텐츠 제한이 적은 고급 모드

## Grok Imagine Spicy Mode 사용 방법

### 1단계: Grok Imagine 접근
1. 즉시 프롬프트 생성을 위해 [GrokPrompts.app](https://grokprompts.app) 방문
2. 또는 X.com에 가서 Grok에 접속
3. "Imagine" 탭으로 이동

### 2단계: Spicy Mode 활성화
- 인터페이스에서 "Spicy Mode" 스위치 켜기
- 참고: spicy mode는 인증된 사용자만 사용 가능

### 3단계: 이 저장소의 프롬프트 사용
- [프롬프트 카테고리](#프롬프트-카테고리) 탐색
- 프롬프트를 복사하여 Grok Imagine에 붙여넣기
- [스타일 수정자](#스타일-수정자-치트시트)를 사용하여 수정
- 생성하고 반복

### 프로 팁:
- 특정 주제로 시작하여 스타일 수정자 추가
- 해상도 힌트 사용: "8k", "4k uhd", "high resolution"
- 여러 스타일 결합: "cinematic lighting, oil painting style"
- 네거티브 프롬프트로 결과 개선 실험

## 스타일 수정자 치트시트

검증된 수정자로 모든 grok imagine prompts를 향상시키세요:

| 카테고리 | 수정자 | 효과 |
|----------|-----------|--------|
| **품질** | 8k uhd, 4k, high resolution, hyperdetailed, intricate details | 이미지 디테일 및 선명도 증가 |
| **조명** | cinematic lighting, dramatic lighting, golden hour, rim lighting, volumetric lighting | 분위기와 분위기 제어 |
| **카메라** | depth of field, bokeh, wide angle, macro shot, telephoto | 카메라 효과 시뮬레이션 |
| **렌더 스타일** | octane render, unreal engine, photorealistic, ray tracing | 전문 렌더링 품질 |
| **예술적** | trending on artstation, award winning, masterpiece, by [artist name] | 예술적 품질 향상 |
| **색상** | vibrant colors, muted tones, black and white, neon colors | 색상 팔레트 제어 |

100개 이상의 수정자를 보려면 [스타일 수정자 참조](./guides/style-modifiers-reference.md)를 참조하세요.

## 프롬프트 공식

일관된 결과를 위해 검증된 공식 사용:

```
[주제] + [동작/포즈] + [환경] + [조명] + [스타일] + [품질 수정자]
```

**예시:**
```
신비로운 숲의 수호자 + 고대 떡갈나무 옆에 서 있는 +
빛나는 버섯이 있는 마법의 숲 + 부드러운 천상의 조명 +
판타지 디지털 아트 스타일 + 8k uhd, hyperdetailed, trending on artstation
```

더 많은 템플릿과 공식은 [프롬프트 공식 가이드](./guides/prompt-formula.md)를 방문하세요.

## FAQ

### grok imagine prompts란 무엇인가요?
Grok imagine prompts는 Grok의 AI가 특정 이미지를 생성하도록 안내하는 텍스트 설명입니다. 간단한 것("산 위의 일몰")부터 여러 수정자가 있는 복잡한 프롬프트까지 다양합니다.

### grok imagine spicy mode는 일반 모드와 어떻게 다른가요?
Spicy mode는 콘텐츠 제한이 적어 더 많은 창의적 자유를 허용합니다. 예술적 표현과 성숙한 주제를 위해 설계되었습니다.

### 더 많은 grok prompts for images를 어디서 찾을 수 있나요?
이 저장소에는 10개 카테고리에 200개 이상의 프롬프트가 포함되어 있습니다. [GrokPrompts.app](https://grokprompts.app)에서 맞춤 프롬프트를 생성할 수도 있습니다.

### 좋은 grok imagine prompt를 만드는 것은 무엇인가요?
좋은 프롬프트는 구체적이고, 스타일 수정자를 사용하며, 품질 키워드(8k, detailed)를 포함하고, 조명/분위기를 지정합니다.

### 이 프롬프트를 상업적으로 사용할 수 있나요?
네, 이 프롬프트는 MIT 라이선스입니다. 생성된 이미지는 Grok의 서비스 약관을 따릅니다.

### 더 나은 결과를 얻으려면 어떻게 해야 하나요?
- 주제와 세부사항에 대해 구체적으로 작성
- 여러 스타일 수정자 사용
- 조명 및 분위기 지정
- 해상도/품질 키워드 포함
- 결과에 따라 반복 및 개선

### 결과를 어디에 공유할 수 있나요?
Reddit(r/xAI)에서 토론에 참여하거나 X에서 #GrokImagine으로 공유하세요. 커뮤니티 기능은 [GrokPrompts.app](https://grokprompts.app)을 방문하세요.
