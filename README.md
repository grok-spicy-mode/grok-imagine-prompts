## What is Grok Imagine?

Grok Imagine is X.AI's powerful image generation model integrated into the Grok platform. It creates high-quality images from text descriptions using advanced AI technology. The platform offers two modes:

- **Normal Mode**: Standard image generation suitable for all audiences
- **Spicy Mode**: Advanced mode with fewer content restrictions for creative expression

## Prompt Formula

Use this proven formula for consistent results:

```
[Subject] + [Action/Pose] + [Environment] + [Lighting] + [Style] + [Quality Modifiers]
```

**Example:**
```
A mystical forest guardian + standing beside ancient oak tree +
enchanted forest with glowing mushrooms + soft ethereal lighting +
fantasy digital art style + 8k uhd, hyperdetailed, trending on artstation
```

## FAQ

### What are grok imagine prompts?
Grok imagine prompts are text descriptions that guide Grok's AI to generate specific images. They can range from simple ("a sunset over mountains") to complex multi-modifier prompts.

### How does grok imagine spicy mode differ from normal mode?
Spicy mode allows more creative freedom with fewer content restrictions. It's designed for artistic expression and mature themes.

### Where can I find more grok prompts for images?
This repository contains 200+ prompts across 10 categories. You can also generate custom prompts at [GrokPrompts.app](https://grokprompts.app).

### What makes a good grok imagine prompt?
Good prompts are specific, use style modifiers, include quality keywords (8k, detailed), and specify lighting/atmosphere.

### Can I use these prompts commercially?
Yes, these prompts are MIT licensed. Generated images follow Grok's terms of service.

### How do I get better results?
- Be specific about subject and details
- Use multiple style modifiers

- Iterate and refine based on results
