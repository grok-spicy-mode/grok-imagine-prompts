## Combining Modifiers

### Effective Stacking

Combine modifiers from different categories for best results:

**Example 1: Cinematic Portrait**
```
Portrait + cinematic lighting + shallow depth of field +
dramatic + 85mm lens + photorealistic + 8k uhd
```

**Example 2: Fantasy Landscape**
```
Fantasy landscape + golden hour + atmospheric +
volumetric lighting + highly detailed + trending on artstation + 8k uhd
```

**Example 3: Product Photography**
```
Product shot + studio lighting + sharp focus +
clean white background + commercial photography + 8k uhd + professional
```

### Modifier Order

While order doesn't strictly matter, this structure works well:

1. **Subject** (what)
2. **Action/Pose** (doing what)
3. **Environment** (where)
4. **Lighting** (how lit)
5. **Style** (art style)
6. **Camera** (technical)
7. **Quality** (resolution/detail)
