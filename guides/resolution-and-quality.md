# Resolution and Quality Guide

Optimize image output quality for Grok Imagine prompts.

## Supported Resolutions

Grok Imagine supports various aspect ratios. Choose based on intended use:

### Standard Resolutions

| Resolution | Aspect Ratio | Best For | Use Cases |
|------------|--------------|----------|-----------|
| **1024x1024** | 1:1 Square | Portraits, product shots, social media | Instagram posts, profile pictures, product photography |
| **1920x1080** | 16:9 Landscape | Landscapes, wallpapers, presentations | Desktop wallpapers, YouTube thumbnails, wide scenes |
| **1080x1920** | 9:16 Portrait | Mobile content, stories | Instagram/TikTok stories, mobile wallpapers |
| **1536x1024** | 3:2 | Photography | Standard photo printing, traditional photography |
| **1024x1536** | 2:3 Portrait | Vertical photos | Portrait orientation photography |

### Choosing the Right Resolution

**For Portraits:**
- Use 1024x1024 for headshots and close-ups
- Use 1024x1536 for full-body portraits
- Use 1080x1920 for mobile-optimized vertical shots

**For Landscapes:**
- Use 1920x1080 for standard wide landscapes
- Use 1536x1024 for traditional photo aspect ratios

**For Products:**
- Use 1024x1024 for square product shots
- Use consistent resolution across product lines

## Quality Keywords

Add these to prompts for enhanced quality:

### Resolution Keywords

| Keyword | Effect | When to Use |
|---------|--------|-------------|
| **8k uhd** | Maximum detail | All high-quality images |
| **4k** | High resolution | Standard professional work |
| **high resolution** | Enhanced clarity | General quality boost |
| **ultra hd** | Maximum sharpness | Critical detail work |

## Optimization Strategies

### 1. Match Resolution to Subject

**Close-ups:**
```
Portrait headshot, 1024x1024, sharp focus, 8k uhd
```

**Wide Scenes:**
```
Mountain landscape, 1920x1080, highly detailed, 8k uhd
```

### 2. Stack Quality Modifiers

Combine 3-5 quality keywords:
```
Professional product photography, 8k uhd, sharp focus,
photorealistic, commercial quality, studio lighting
```

### 3. Specify Detail Areas

```
Hyperdetailed face, sharp eyes, intricate hair texture,
8k uhd portrait photography
```

## Platform-Specific Optimization

### Instagram

**Feed Posts:**
- Resolution: 1024x1024 (square) or 1080x1350 (portrait)
- Keywords: vibrant colors, sharp, high quality

**Stories:**
- Resolution: 1080x1920
- Keywords: mobile-optimized, vertical format

### YouTube

**Thumbnails:**
- Resolution: 1920x1080
- Keywords: bold, high contrast, attention-grabbing

### Print

**Standard Prints:**
- Resolution: 1536x1024 or higher
- Keywords: 8k uhd, print quality, photorealistic

**Large Format:**
- Use highest resolution available
- Keywords: hyperdetailed, ultra hd, museum quality

## Common Quality Issues

### Issue: Blurry Results

**Solutions:**
- Add "sharp focus" and "8k uhd"
- Remove motion blur keywords
- Specify "crystal clear" and "tack sharp"

### Issue: Low Detail

**Solutions:**
- Add "hyperdetailed" and "intricate details"
- Increase resolution specification
- Add "highly detailed" throughout prompt

### Issue: Artificial Looking

**Solutions:**
- Add "photorealistic" and "natural"
- Specify realistic lighting
- Remove overly stylized keywords

### Issue: Poor Composition

**Solutions:**
- Specify composition rules (rule of thirds, golden ratio)
- Add "professional composition"
- Reference specific camera angles

## Quality Checklist

Before submitting your prompt, verify:

- [ ] Resolution specified or implied
- [ ] At least one quality keyword (8k uhd, photorealistic, etc.)
- [ ] Detail level specified (hyperdetailed, sharp focus, etc.)
- [ ] Lighting defined
- [ ] Style clearly indicated
- [ ] Subject properly described

## Resolution vs File Size

Note: Grok Imagine output file sizes are optimized regardless of specified resolution keywords. Quality keywords affect rendering quality, not necessarily file size.

## Testing and Iteration

**Quality Test Method:**
1. Generate with basic prompt
2. Add one quality keyword
3. Compare results
4. Stack multiple quality keywords
5. Find optimal combination

**A/B Testing Examples:**

**Test 1: Basic vs Enhanced**
- A: "Mountain landscape"
- B: "Mountain landscape, 8k uhd, photorealistic"

**Test 2: Detail Levels**
- A: "Portrait, detailed"
- B: "Portrait, hyperdetailed, sharp focus, 8k uhd"
