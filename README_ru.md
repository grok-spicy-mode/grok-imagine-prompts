## Что такое Grok Imagine?

Grok Imagine — это мощная модель генерации изображений от X.AI, интегрированная в платформу Grok. Она создаёт высококачественные изображения из текстовых описаний, используя передовые технологии ИИ. Платформа предлагает два режима:

- **Обычный Режим**: Стандартная генерация изображений, подходящая для всех аудиторий
- **Spicy Mode**: Продвинутый режим с меньшими ограничениями контента для творческого самовыражения

## Как Использовать Grok Imagine Spicy Mode

### Шаг 1: Доступ к Grok Imagine
1. Посетите [GrokPrompts.app](https://grokprompts.app) для мгновенной генерации промптов
2. Или перейдите на X.com и войдите в Grok
3. Перейдите на вкладку "Imagine"

### Шаг 2: Активация Spicy Mode
- Включите переключатель "Spicy Mode" в интерфейсе
- Примечание: spicy mode доступен только для верифицированных пользователей

### Шаг 3: Использование Промптов из Этого Репозитория
- Просмотрите наши [категории промптов](#категории-промптов)
- Скопируйте любой промпт и вставьте в Grok Imagine
- Модифицируйте, используя наши [модификаторы стиля](#шпаргалка-по-модификаторам-стиля)
- Генерируйте и итерируйте

### Профессиональные Советы:
- Начните с конкретных объектов и добавьте модификаторы стиля
- Используйте указатели разрешения: "8k", "4k uhd", "high resolution"
- Комбинируйте несколько стилей: "cinematic lighting, oil painting style"
- Экспериментируйте с негативными промптами для уточнения результатов

## Шпаргалка по Модификаторам Стиля

Улучшите любые grok imagine prompts этими проверенными модификаторами:

| Категория | Модификаторы | Эффект |
|----------|-----------|--------|
| **Качество** | 8k uhd, 4k, high resolution, hyperdetailed, intricate details | Увеличивает детализацию и чёткость изображения |
| **Освещение** | cinematic lighting, dramatic lighting, golden hour, rim lighting, volumetric lighting | Управляет настроением и атмосферой |
| **Камера** | depth of field, bokeh, wide angle, macro shot, telephoto | Имитирует эффекты камеры |
| **Стиль Рендеринга** | octane render, unreal engine, photorealistic, ray tracing | Профессиональное качество рендеринга |
| **Художественный** | trending on artstation, award winning, masterpiece, by [artist name] | Повышение художественного качества |
| **Цвет** | vibrant colors, muted tones, black and white, neon colors | Управление цветовой палитрой |

## Формула Промптов

Используйте эту проверенную формулу для стабильных результатов:

```
[Объект] + [Действие/Поза] + [Окружение] + [Освещение] + [Стиль] + [Модификаторы Качества]
```

**Пример:**
```
Мистический хранитель леса + стоящий возле древнего дуба +
заколдованный лес со светящимися грибами + мягкое эфирное освещение +
стиль фэнтезийного цифрового искусства + 8k uhd, hyperdetailed, trending on artstation
```

Для большего количества шаблонов и формул посетите наше [Руководство по Формулам Промптов](./guides/prompt-formula.md).

## FAQ

### Что такое grok imagine prompts?
Grok imagine prompts — это текстовые описания, которые направляют ИИ Grok для генерации конкретных изображений. Они могут варьироваться от простых («закат над горами») до сложных промптов с множественными модификаторами.

### Чем grok imagine spicy mode отличается от обычного режима?
Spicy mode позволяет больше творческой свободы с меньшими ограничениями контента. Он разработан для художественного самовыражения и зрелых тем.

### Где я могу найти больше grok prompts for images?
Этот репозиторий содержит более 200 промптов в 10 категориях. Вы также можете генерировать пользовательские промпты на [GrokPrompts.app](https://grokprompts.app).

### Что делает хороший grok imagine prompt?
Хорошие промпты специфичны, используют модификаторы стиля, включают ключевые слова качества (8k, detailed) и указывают освещение/атмосферу.

### Могу ли я использовать эти промпты коммерчески?
Да, эти промпты лицензированы по MIT. Сгенерированные изображения следуют условиям обслуживания Grok.

### Как получить лучшие результаты?
- Будьте конкретны об объекте и деталях
- Используйте несколько модификаторов стиля
- Указывайте освещение и атмосферу
- Включайте ключевые слова разрешения/качества
- Итерируйте и уточняйте на основе результатов

### Где я могу поделиться своими результатами?
Присоединяйтесь к обсуждениям на Reddit (r/xAI) или делитесь в X с хэштегом #GrokImagine. Посетите [GrokPrompts.app](https://grokprompts.app) для функций сообщества.
