## ¿Qué es Grok Imagine?

Grok Imagine es el potente modelo de generación de imágenes de X.AI integrado en la plataforma Grok. Crea imágenes de alta calidad a partir de descripciones de texto utilizando tecnología de IA avanzada. La plataforma ofrece dos modos:

- **Modo Normal**: Generación de imágenes estándar adecuada para todos los públicos
- **Spicy Mode**: Modo avanzado con menos restricciones de contenido para expresión creativa

## Cómo Usar Grok Imagine Spicy Mode

### Paso 1: Acceder a Grok Imagine
1. Visita [GrokPrompts.app](https://grokprompts.app) para generación instantánea de prompts
2. O ve a X.com y accede a Grok
3. Navega a la pestaña "Imagine"

### Paso 2: Activar Spicy Mode
- Activa el interruptor "Spicy Mode" en la interfaz
- Nota: spicy mode solo está disponible para usuarios verificados

### Paso 3: Usar Prompts de Este Repositorio
- Explora nuestras [categorías de prompts](#categorías-de-prompts)
- Copia cualquier prompt y pégalo en Grok Imagine
- Modifica usando nuestros [modificadores de estilo](#hoja-de-trucos-de-modificadores-de-estilo)
- Genera e itera

### Consejos Profesionales:
- Comienza con sujetos específicos y añade modificadores de estilo
- Usa indicadores de resolución: "8k", "4k uhd", "high resolution"
- Combina múltiples estilos: "cinematic lighting, oil painting style"
- Experimenta con prompts negativos para refinar resultados

## Hoja de Trucos de Modificadores de Estilo

Mejora cualquier grok imagine prompts con estos modificadores probados:

| Categoría | Modificadores | Efecto |
|----------|-----------|--------|
| **Calidad** | 8k uhd, 4k, high resolution, hyperdetailed, intricate details | Aumenta el detalle y nitidez de la imagen |
| **Iluminación** | cinematic lighting, dramatic lighting, golden hour, rim lighting, volumetric lighting | Controla el ambiente y atmósfera |
| **Cámara** | depth of field, bokeh, wide angle, macro shot, telephoto | Simula efectos de cámara |
| **Estilo de Renderizado** | octane render, unreal engine, photorealistic, ray tracing | Calidad de renderizado profesional |
| **Artístico** | trending on artstation, award winning, masterpiece, by [artist name] | Mejora la calidad artística |
| **Color** | vibrant colors, muted tones, black and white, neon colors | Control de paleta de colores |

Consulta nuestra [Referencia Completa de Modificadores de Estilo](./guides/style-modifiers-reference.md) para más de 100 modificadores.

## Fórmula de Prompts

Usa esta fórmula probada para resultados consistentes:

```
[Sujeto] + [Acción/Pose] + [Entorno] + [Iluminación] + [Estilo] + [Modificadores de Calidad]
```

**Ejemplo:**
```
Un guardián místico del bosque + de pie junto a un roble antiguo +
bosque encantado con hongos brillantes + iluminación etérea suave +
estilo de arte digital fantástico + 8k uhd, hyperdetailed, trending on artstation
```

Para más plantillas y fórmulas, visita nuestra [Guía de Fórmulas de Prompts](./guides/prompt-formula.md).

## FAQ

### ¿Qué son los grok imagine prompts?
Los grok imagine prompts son descripciones de texto que guían a la IA de Grok para generar imágenes específicas. Pueden variar desde simples ("una puesta de sol sobre montañas") hasta prompts complejos con múltiples modificadores.

### ¿En qué se diferencia grok imagine spicy mode del modo normal?
Spicy mode permite más libertad creativa con menos restricciones de contenido. Está diseñado para expresión artística y temas maduros.

### ¿Dónde puedo encontrar más grok prompts for images?
Este repositorio contiene más de 200 prompts en 10 categorías. También puedes generar prompts personalizados en [GrokPrompts.app](https://grokprompts.app).

### ¿Qué hace que un grok imagine prompt sea bueno?
Los buenos prompts son específicos, usan modificadores de estilo, incluyen palabras clave de calidad (8k, detailed) y especifican iluminación/atmósfera.

### ¿Puedo usar estos prompts comercialmente?
Sí, estos prompts tienen licencia MIT. Las imágenes generadas siguen los términos de servicio de Grok.

### ¿Cómo obtengo mejores resultados?
- Sé específico sobre el sujeto y los detalles
- Usa múltiples modificadores de estilo
- Especifica iluminación y atmósfera
- Incluye palabras clave de resolución/calidad
- Itera y refina según los resultados

### ¿Dónde puedo compartir mis resultados?
Únete a las discusiones en Reddit (r/xAI) o comparte en X con #GrokImagine. Visita [GrokPrompts.app](https://grokprompts.app) para funciones comunitarias.
