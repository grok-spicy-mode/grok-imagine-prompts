# Quality Modifiers Cheat Sheet

Essential quality and detail keywords for grok imagine prompts.

## Resolution Keywords

| Keyword | Effect | When to Use |
|---------|--------|-------------|
| **8k uhd** | Ultra high definition | All high-quality images |
| **4k** | High resolution | Standard quality work |
| **high resolution** | Enhanced clarity | General use |
| **ultra hd** | Maximum detail | Critical work |
| **hd** | Good quality | Basic enhancement |

**Usage:** Add at least one to every prompt
```
"portrait, 8k uhd"
```

## Quality Tier System

### Tier 1: Essential (Always Include)
- 8k uhd (or 4k)
- One detail keyword (detailed, hyperdetailed)
- One quality keyword (photorealistic, professional)

### Tier 2: Enhanced (Add for Better Results)
- Specific detail level (intricate details, highly detailed)
- Professional qualification (commercial quality, award winning)
- Style-specific quality (trending on artstation for digital art)

### Tier 3: Maximum (Premium Results)
- Stack multiple detail keywords
- Add excellence keywords (masterpiece, museum quality)
- Include technical specifics (ray tracing, octane render)

## Testing Quality Keywords

**Method:** Generate same subject with different quality stacks

**Test 1: Basic vs Enhanced**
- Basic: "portrait, 8k"
- Enhanced: "portrait, 8k uhd, photorealistic, sharp focus, professional"

**Test 2: Detail Levels**
- Low: "landscape, detailed"
- High: "landscape, hyperdetailed, intricate details, highly detailed, 8k uhd"

**Test 3: Professional vs Artistic**
- Professional: "scene, professional, commercial quality, 8k uhd"
- Artistic: "scene, masterpiece, award winning, gallery worthy, 8k uhd"
