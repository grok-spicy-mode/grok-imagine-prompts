## Grok Imagineとは？

Grok ImageはGrokプラットフォームに統合されたX.AIの強力な画像生成モデルです。高度なAI技術を使用してテキストの説明から高品質な画像を作成します。プラットフォームは2つのモードを提供します：

- **ノーマルモード**：すべての視聴者に適した標準的な画像生成
- **Spicy Mode**：クリエイティブな表現のためのコンテンツ制限が少ない高度なモード

## Grok Imagine Spicy Modeの使い方

### ステップ1：Grok Imagineへアクセス
1. 即座にプロンプト生成するために[GrokPrompts.app](https://grokprompts.app)を訪問
2. またはX.comにアクセスしてGrokに入る
3. 「Imagine」タブに移動

### ステップ2：Spicy Modeを有効化
- インターフェースで「Spicy Mode」スイッチをオンにする
- 注意：spicy modeは認証済みユーザーのみ利用可能

### ステップ3：このリポジトリのプロンプトを使用
- [プロンプトカテゴリ](#プロンプトカテゴリ)を閲覧
- 任意のプロンプトをコピーしてGrok Imagineに貼り付け
- [スタイル修飾子](#スタイル修飾子チートシート)を使用して変更
- 生成して反復

### プロのヒント：
- 具体的な主題から始めてスタイル修飾子を追加
- 解像度ヒントを使用：「8k」、「4k uhd」、「high resolution」
- 複数のスタイルを組み合わせる：「cinematic lighting, oil painting style」
- ネガティブプロンプトで結果を洗練する実験

## スタイル修飾子チートシート

実証済みの修飾子で任意のgrok imagine promptsを強化：

| カテゴリ | 修飾子 | 効果 |
|----------|-----------|--------|
| **品質** | 8k uhd, 4k, high resolution, hyperdetailed, intricate details | 画像の詳細と鮮明さを向上 |
| **照明** | cinematic lighting, dramatic lighting, golden hour, rim lighting, volumetric lighting | ムードと雰囲気を制御 |
| **カメラ** | depth of field, bokeh, wide angle, macro shot, telephoto | カメラ効果をシミュレート |
| **レンダースタイル** | octane render, unreal engine, photorealistic, ray tracing | プロフェッショナルなレンダリング品質 |
| **芸術的** | trending on artstation, award winning, masterpiece, by [artist name] | 芸術的品質を向上 |
| **色** | vibrant colors, muted tones, black and white, neon colors | カラーパレット制御 |

100以上の修飾子については、完全な[スタイル修飾子リファレンス](./guides/style-modifiers-reference.md)を参照してください。

## プロンプト公式

一貫した結果のためにこの実証済みの公式を使用：

```
[主題] + [アクション/ポーズ] + [環境] + [照明] + [スタイル] + [品質修飾子]
```

**例：**
```
神秘的な森の守護者 + 古代のオークの木の横に立っている +
光る茸のある魔法の森 + 柔らかな幻想的な照明 +
ファンタジーデジタルアートスタイル + 8k uhd, hyperdetailed, trending on artstation
```

より多くのテンプレートと公式については、[プロンプト公式ガイド](./guides/prompt-formula.md)を訪問してください。

## よくある質問

### grok imagine promptsとは何ですか？
Grok imagine promptsは、Grokのアイが特定の画像を生成するように導くテキストの説明です。シンプルなもの（「山々の上の夕日」）から複数の修飾子を持つ複雑なプロンプトまで範囲があります。

### grok imagine spicy modeはノーマルモードとどう違いますか？
Spicy modeはコンテンツ制限が少なく、より多くのクリエイティブな自由を許可します。芸術的表現と成熟したテーマのために設計されています。

### より多くのgrok prompts for imagesをどこで見つけられますか？
このリポジトリには10のカテゴリに200以上のプロンプトが含まれています。[GrokPrompts.app](https://grokprompts.app)でカスタムプロンプトを生成することもできます。

### 良いgrok imagine promptを作るものは何ですか？
良いプロンプトは具体的で、スタイル修飾子を使用し、品質キーワード（8k、detailed）を含み、照明/雰囲気を指定します。

### これらのプロンプトを商業的に使用できますか？
はい、これらのプロンプトはMITライセンスです。生成された画像はGrokの利用規約に従います。

### より良い結果を得るには？
- 主題と詳細について具体的に
- 複数のスタイル修飾子を使用
- 照明と雰囲気を指定
- 解像度/品質キーワードを含める
- 結果に基づいて反復して洗練

### 結果をどこで共有できますか？
Reddit（r/xAI）でディスカッションに参加するか、Xで#GrokImagineで共有してください。コミュニティ機能については[GrokPrompts.app](https://grokprompts.app)を訪問してください。
